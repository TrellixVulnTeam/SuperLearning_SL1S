\subsection{Question 3}\label{question-3}}

FizzBuzz is a classic programming exercise. Iterate over the numbers
from 0 to 99 and for each number:

\begin{itemize}
\item
  If the number is divisible by 3, print Fizz
\item
  If the number is divisible by 5, print Buzz
\item
  If the number is divisible by both 3 and 5, print Fizz Buzz
\item
  Otherwise, just print the number.
\end{itemize}

\textbf{Hint:} The modulo operator which gives you a division remainder
is the percentage sign, \%. For example:
